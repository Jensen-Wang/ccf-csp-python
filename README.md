# ccf-csp-python
ccf考试历届真题python语言解答（持续更新中）
### Tips
* eval()很好用，但是作为输入效率比较低
* collections与heapq等库的数据结构很好用
* python递归层数极限为1000，可使用import sys sys.setrecursionlimit(100000)提升到4000左右，再高需要改为非递归算法
